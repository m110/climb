# climb
Python library for interactive command line applications.

## Featuring:

* Interactive and batch mode.
* Tab-completions out of the box.
* Ready to handle directory paths.

## Installation

```
pip3 install climb
```

or get the source and run:

```
python3 setup.py install
```
